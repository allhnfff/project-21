FAQ

Berapa lama prosesnya?

Biasanya cepat setelah konfirmasi order di WhatsApp. Untuk jasa desain/joki, tergantung tingkat kesulitan & deadline.

Garansi ada?

Ada untuk beberapa produk sesuai ketentuan. Tanyakan admin untuk detail garansi per produk.

Metode pembayaran?

Silakan chat admin untuk info metode pembayaran yang tersedia.

Akun sharing/private maksudnya apa?

Sharing = dipakai bersama sesuai paket. Private = khusus kamu. Detailnya bisa berbeda per layanan—konfirmasi ke admin.

Cara order lewat Bot gimana?
